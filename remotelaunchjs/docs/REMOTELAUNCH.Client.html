Class: Client

REMOTELAUNCH. Client

new Client(options)

A Client interacts with a RemoteLaunchServer, showing information on different launch files previously configured in a config file, and calling services to start and stop those launch files. Emits the following events: * 'list_update' - there was an update in the file list * 'fileStart' - a process has started or was running when page was loaded - sends the file name as an argument * 'fileStop' - a process has stopped - sends the file name as an argument
Parameters:
Name Type Description
options object with following keys: * ros - the ROSLIB.Ros connection handle * topic (optional) - the topic with information on launch files (default: '/list_launch_files') * serverName (optional) - the server node name, used for namespaces (default: '/remote_launch_server')
